An \code{EventPreSyscall} is triggered when a thread enters a system call,
provided that the thread is in system call tracing mode.

Associated EventType Code: \code{Syscall}

\begin{apient}
Dyninst::Address getAddress() const
\end{apient}

\apidesc{
This function returns the address where the system call occurred.
}

\begin{apient}
MachSyscall getSyscall() const
\end{apient}

\apidesc{
This function returns information about the system call.
}
